What is Docker Hub?

Docker Hub is a cloud-based registry service that allows developers to link code repositories, build their applications, test them, and distribute container images. It's the world's largest library and community for container images.

Types of Docker Repositories

🌍
Public

Public Repositories

Free & Open Access

  • Accessible to anyone on the internet
  • Unlimited pulls for public images
  • Ideal for open-source projects
  • Free to create and maintain
🔒
Private

Private Repositories

Secure & Controlled

  • Restricted to authorized users
  • Secure proprietary images
  • Team & organization management
  • Requires paid subscription for more
✅
Official

Official Images

Verified by Docker

  • Maintained by Docker & partners
  • Security-scanned regularly
  • Best practices applied
  • Examples: nginx, ubuntu, python

Docker Push & Pull Workflow

01

Build Image

docker build -t myapp:1.0 .
02

Login to Hub

docker login
03

Tag Image

docker tag myapp:1.0 username/myapp:1.0
04

Push Image

docker push username/myapp:1.0
05

Pull Image

docker pull username/myapp:1.0
